ODYSSEY-Y-LUNA is an Android application that allows users to visualize maps, track routes, and store journey data using Baato Maps API and MapLibre SDK.  
The project follows the MVVM architecture to ensure clean code separation and scalability.

---
Features

-  Map visualization using Baato raster tiles
-  Real-time route tracking
-  Polyline route drawing
-  Store route data in database
-  MVVM architecture
-  Location tracking using Android Location Services

---
Technologies Used

- Language: Kotlin
- Architecture: MVVM (Model-View-ViewModel)
- Map SDK: MapLibre Android SDK
- Map API: Baato Maps API
- UI: Jetpack Compose / XML (depending on your project)

---
Setup

To setup this project clone the repository "https://github.com/nrishshrestha/ODYSSEY-Y-LUNA-" then run the project in android studio. Launch the application in the android studio emulator or any android device.
